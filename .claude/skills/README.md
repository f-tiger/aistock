# Project Skills

以下 skills 来自 [ECC (Everything Claude Code)](https://github.com/affaan-m/ECC)(MIT 许可,见 ECC-LICENSE,安装自 commit 4092795),经安全审查后择优安装(纯 Markdown、无脚本/hooks;排除了依赖第三方付费服务的 social-publisher):

| Skill | 用途(服务本站增长自动化) |
|---|---|
| seo | 技术 SEO 审计与关键词映射——程序化 vs 页与站点优化 |
| content-engine | 平台原生内容体系——周报/发帖的质量标准 |
| crosspost | 多平台分发适配——雪球/小红书/X/Reddit 各写各的 |
| article-writing | 长文写作——避免 AI 味,保持真实观点 |
| marketing-campaign | 发布战役编排——launch/冲刺的方法论 |
| brand-voice | 品牌声音档案——全渠道文案一致性 |
| market-research | 市场调研规范——带出处、面向决策 |

这些技能对本仓库的所有 Claude 会话生效,包括每周自动内容运营(ops-weekly)。
